## 二、代码增强（Code Enhancement）— 逐模块清单

"代码增强"指在 Baseline 基础上对**模型结构、训练策略或工程质量**做出的改进。

### 2.1 模型结构增强

- [x] **`experiments/attention/pointnet.py`** — SE-Block 通道注意力
  - 在 PointNet 全局特征（1024 维）之后插入 `SEBlock` 模块
  - 通过 Squeeze-and-Excitation 机制自适应地对各通道加权
  - 含 `BatchNorm` + `Dropout(0.3)` 正则化，提升训练稳定性
  - **增强点**：`SEBlock` 类、`PointNetWithSE.se` 属性及其 `forward` 中的 `self.se(x)` 调用

### 2.2 数据层增强

- [x] **`experiments/augmentation/dataset.py`** — 训练期数据随机变换
  - `random_rotate_point_cloud`：绕 Y 轴随机旋转，增强旋转不变性
  - `jitter_point_cloud`：高斯抖动加噪（σ=0.01，截断±0.05），模拟点云扫描噪声
  - **增强点**：两个函数在数据加载阶段（DataLoader）调用，属于数据层代码增强

### 2.3 工程与流程增强

- [x] **`colab_final/colab_setup.sh`** — 可复现性增强
  - 一键克隆依赖仓库、安装环境、下载数据集，降低手动操作出错概率
- [x] **`colab_final/train_baseline.sh`** — 基线流程标准化
  - 固化 PointNet Baseline 训练命令，便于统一复现
- [x] **`colab_final/train_dgcnn.sh`** — 对比实验标准化
  - 固化 DGCNN 对比实验入口，方便与 Baseline 公平比较
- [x] **`colab_final/README.md`** — 文档增强
  - 明确各脚本执行顺序，提高交付可读性
